# AI-movie

AI-movie 是一个为漫剧和长视频生成设计的**智能 AI 提示词工作流与约束优化应用**，基于最新 Vue 3 + Vite 现代前端架构。

该项目着眼于解决 AI 生成动画/漫剧过程中的核心痛点：**字幕污染**、**上下文断层** 和 **地理跳脱**。

## 🌟 核心特性

- **独创防篡改字幕隔离**：内部系统级双重限定输出约束词汇（`--no text, --no subtitles`等），确保您拿到生图/视频工具中的结果 100% 毫无文本水印。
- **动态记忆链与防跳脱引擎**：解决长视频生成中的“翻脸”和“瞬移”问题。内置上下文 Memory 记录机制，并强制 AI 确保“当前镜头第一帧与上一镜头最后一帧地理位置、衣服穿搭”达到严丝合缝的连贯。
- **长集剧本专属智能解析**：您可以将至多一集的剧本直接放进来由 AI 解析。工具不仅能提取可用场景文本，还能利用红线剔红标注将旁白、无法被画出的技术性文字立刻拦截。
- **现代化生产力面板**：极简光面的工作台与内置定制包括韩漫彩漫、赛博朋克、现代都市在内的顶级主流风预设。

## 🚀 运行项目

本项目为纯前端轻量化开发，需要您本地拥有 `Node.js`。

1. **环境安装**
   克隆本项目后，安装依赖：
   ```bash
   npm install
   ```

2. **环境变量配置**
   请在项目根目录下查看或编辑 `.env` 配置文件（用于确保您的 KEY 不被暴露在公网）：
   ```env
   VITE_API_URL=https://api.moonshot.cn/v1
   VITE_API_KEY=您的_API_KEY
   ```

3. **启动引擎**
   ```bash
   npm run dev
   ```
   随后在浏览器打开控制台中打印出的本地网址（通常为 `http://localhost:5173`）开启体验。
